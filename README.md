# Ecommerce-WEB - Fullstack Spring + Angular
***
### Ecommerce web com Spring Boot no backend e Angular no frontend

- Backend:
  - Java 
  - Spring boot
  - Spring Data Rest
  - Spring Data JPA
  - Connector Mysql

- Banco de dados:
  - MySql
  
- Frontend:
  - Html
  - CSS
  - Bootstrap
  - Javascript
  - Typescript
  - Angular
